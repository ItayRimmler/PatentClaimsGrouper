Welcome to PatentClaimsGrouper (PCG). The name speaks for itself. It groups patent claims...

The program runs on python version 3.7.16, uses spacy 2.1.0's "en_core_web_sm" model, and a neuralcoef pipeline.

Commands that were used in python console are: nltk.download('stopwords'), nltk.download('omw-1.4'), nltk.download('wordnet'), nltk.download('punkt')
